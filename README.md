# 🏢 Office Management System (OMS Portal)

A premium, full-stack Office Management System built with **Angular** and **Node.js/Express**.

🚀 **Live Demo:** [https://office-management-system-eta.vercel.app](https://office-management-system-eta.vercel.app)

---

## ✨ Features

- **🎨 Dynamic Theming**: 5 pre-built color themes with Dark and Light mode support.
- **📱 Responsive Design**: Fully collapsible sidebar and mobile-friendly layouts.
- **🎬 Rich Animations**: Micro-animations, staggered entrances, and smooth transitions.
- **👥 User Management**: Role-based access (Admin, Manager, Employee) with strict authentication.
- **📊 Real-time Stats**: Track company performance, attendance, and project progress.
- **📅 Leave & Attendance**: Automated leave calculations and attendance tracking.
- **📁 Document Management**: Centralized file storage for company assets.
- **🤝 Team Collaboration**: Manage departments and teams effectively.
- **✅ Task Board**: Assigned tasks with status tracking.

## 🛠️ Technology Stack

- **Frontend**: Angular 17, Vanilla CSS (Premium Custom Design)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas (Cloud)
- **Deployment**: Vercel (Front-end & Backend API)
- **Auth**: JWT (JSON Web Tokens) with Secure Middleware

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ajith-softwaredeveloper/office-management-system.git
   ```

2. **Frontend Setup**
   ```bash
   cd angular-app
   npm install
   npm start
   ```

3. **Backend Setup**
   ```bash
   cd server
   npm install
   npm start
   ```

*Make sure to configure your `.env` with `MONGODB_URI` and `JWT_SECRET`.*
